## Adding Media

When adding media assets:

1. **Optimize file sizes** - Compress images, use appropriate resolutions
2. **Remove sensitive data** - No real account info, API keys, etc.
3. **Use descriptive names** - `dashboard-overview.png`, not `screenshot1.png`
4. **Add captions** - Update this README with descriptions
